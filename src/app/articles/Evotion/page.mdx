Evotion is an AI chat platform that brings together 30+ of the best language models in one place.
The core idea is simple: instead of juggling multiple subscriptions and switching between tabs, users get access to ChatGPT, Claude, Gemini, Grok, and many more through a single unified interface at the best price.

<Image src={evotionHero} alt="Evotion platform overview" />

## Fusion Mode

The most distinctive feature of Evotion is its **Fusion Mode**, a system that automatically selects the best model for each query and, when beneficial, runs multiple models in parallel and synthesizes their outputs into a single refined response.

Instead of needing to know which model excels at code, which is best for creative writing, and which handles reasoning tasks best, Fusion Mode handles that decision automatically. The result is consistently higher-quality responses without requiring the user to manage model selection manually.

<Image src={evotionModels} alt="Evotion model selection and fusion mode" />

## Features

- Access to 30+ language models (ChatGPT, Claude, Gemini, Grok, Mistral, and more) through a single interface.

- Fusion Mode that automatically selects and combines models based on the type of query.

- Multi-modal support: send text, images, and documents (PDFs) in the same conversation.

- Real-time usage tracking with precise cost calculations per message and per session.

- Web search integration that grounds responses in up-to-date information.

- Artifact generation for documents, code, and structured outputs.

- Subscription and pay-per-use billing managed through Stripe.

- Device fingerprinting, MFA support, rate limiting, and audit logging for security.

<Image src={evotionChat} alt="Evotion chat interface" />

## Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS.
- **Backend**: tRPC, Next.js API routes, PostgreSQL, Drizzle ORM.
- **AI Integration**: Vercel AI SDK with adapters for all major model providers.
- **Authentication**: NextAuth.js with multiple OAuth providers and MFA.
- **Payment System**: Stripe (subscriptions and pay-per-use metering).
- **Real-time Streaming**: Server-Sent Events for live model response streaming.
- **Monorepo**: Turborepo for managing the frontend and backend packages.
